# CVER

A simple commandline app for searching and looking up opensource vulnerabilities

# Установка

## Через Pip

```bash
  $ pip install cver
```

## Вручную

```bash
  $ git clone https://github.com/citguru/cevr
  $ cd cver
  $ python setup.py install
```
# Использование

```bash
$ cver
```

## Поиск
`search <keyword>`

```bash
$ cver search python
```
## Просмотр

`search <name>`

```bash
$ cver look-up CVE-2020-2121
```
